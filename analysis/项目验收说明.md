# 项目验收说明

## 1. 项目名称

智能交通路口管理车牌识别算法升级项目

## 2. 工单编号

人工智能 CV-智能交通路口管理-车辆、行人目标检测与跟踪算法升级任务

## 3. 项目目标

本项目面向智能交通路口场景，完成车牌识别算法从原有路线向 `YOLOv11 + OCR` 路线的升级，目标包括：
- 提升车牌定位精度
- 提升车牌字符识别精度
- 满足较快响应时间要求
- 为多省份、多场景、多车牌类型扩展提供基础

## 4. 项目方案

当前采用两阶段方案：

1. 使用 `YOLOv11` 完成车牌检测
2. 使用 `OCR(CRNN + CTC)` 完成车牌字符识别

整体流程：
- 输入车辆图像
- 检测车牌区域
- 裁剪车牌图像
- OCR 识别汉字、字母、数字字符序列
- 输出完整车牌号
